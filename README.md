## 📌 Overview
This project builds a Speech Emotion Recognition (SER) system that classifies human emotions from audio signals using deep learning.

## 🎯 Objective
To classify speech into 8 emotions:
- Happy
- Sad
- Angry
- Fearful
- Disgust
- Surprised
- Neutral
- Calm

---

## 🧠 Model Architecture
- CNN (feature extraction)
- Bidirectional LSTM (temporal learning)
- Attention Mechanism (focus on important parts)

---

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Librosa (audio processing)
- Matplotlib

---

## 📊 Dataset
- RAVDESS
- SAVEE

---

## 🔍 Feature Extraction
- MFCC (Mel-Frequency Cepstral Coefficients)
- Mel Spectrograms

---

## 🔁 Data Augmentation
- Noise Injection
- Pitch Shifting
- Time Stretching

---

## 📈 Results
- Accuracy: **87.42%**
- Evaluated using:
  - Confusion Matrix
  - Precision, Recall, F1-score
